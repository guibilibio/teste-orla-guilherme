Visão Geral

A aplicação é baseada na arquitetura MVC (Model-View-Controller) e foi implementada em Java 11+ com Spring Boot. O sistema permite cadastrar Funcionários e Projetos, além de listar Funcionários com seus Projetos e Projetos com seus Funcionários. O relacionamento entre Funcionários e Projetos é muitos para muitos, e a persistência dos dados é feita em um banco de dados relacional.

Tecnologias Utilizadas

Java 11+

Spring Boot (para desenvolvimento de APIs RESTful)

JPA/Hibernate (para persistência no banco de dados)

Banco de Dados Relacional (por exemplo, MySQL, PostgreSQL)

JUnit 5 e Mockito (para testes unitários)

Maven (para gerenciamento de dependências)
